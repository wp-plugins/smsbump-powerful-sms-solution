== Description ==

Powerful extension that allows you to send SMS, MMS, USSD and Voice messages to your customers effortlessly.

An online demo can be found at http://smsbump.wp.isenselabs.com

== Installation ==

1. Upload the `smsbump` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is SMSBump? =

SMSBump is a cloud-based reliable interface for sending SMS, Voice (VMS), MMS and USSD messages to 200+ networks around the world.

= How to create an API key? =

If you want to use the plugin for SMSBump, you need to create an API key. You can do this by signing up here. The account creation is free and gives you credit to start.
